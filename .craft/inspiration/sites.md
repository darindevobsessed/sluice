# Inspiration Sites

## Primary Reference

### LoopOps
- **URL**: https://www.loopops.com/
- **Captured**: 2026-02-03
- **Why**: Clean SaaS landing page, modern design patterns, good typography hierarchy

**Key Elements:**
- Pill-shaped CTA buttons
- Rounded cards with generous padding (24px radius)
- Soft gradient backgrounds (sky/sunset imagery)
- Two-column feature layouts
- Sticky navigation
- Clean sans-serif typography (Waldenburg)
- Muted color palette with accent pops

**Adapted For Gold Miner:**
- Kept: Card patterns, spacing, layout approach
- Changed: Primary color from magenta to rich green (#059669)
- Changed: Typography to Inter (more widely available)
