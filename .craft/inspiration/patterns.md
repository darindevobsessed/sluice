# UI Patterns

## Navigation
- Sticky header with blur backdrop
- Logo left, actions right
- Pill-shaped primary CTA
- Clean, minimal links

## Hero Sections
- Large headline (48px+, semibold)
- Supporting text in muted color
- Gradient or image backgrounds
- Clear CTA placement

## Cards
- Generous border-radius (16-24px)
- Ample padding (24-32px)
- Subtle shadows or borders
- Accent background colors for variety

## Feature Sections
- Two-column layouts on desktop
- Image + text pairings
- Alternating left/right alignment
- Numbered steps for processes

## Typography Hierarchy
- H1: 48px, semibold, tight leading
- H2: 36px, semibold
- H3: 24px, semibold
- Body: 16px, normal, relaxed leading
- Small: 14px, muted color

## Spacing
- Sections: 64-96px vertical padding
- Cards: 24-32px internal padding
- Elements: 16-24px gaps
- Generous whitespace throughout

## Buttons
- Primary: Filled, pill-shaped, bold text
- Secondary: Outlined, pill-shaped
- Hover: Darken primary, add shadow

## Forms
- Rounded inputs matching card radius
- Clear labels above fields
- Inline validation feedback
- Generous touch targets
